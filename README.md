# Code Challenge
* Programming Language - TypeScript
* Development environment - Node 16.13.2

## Running the application
1. Install packages

```
npm install
```

2. Build the project

```
npm run build
```

3. Start the application

```
npm run start
```


## Testing

1. Use eslint to check the code 
```
npm run lint
```

2. Run test cases 
```
npm run test
```
